# PixTech App

Welcome to PixTech App, a modern Point of Sale (POS) and Service Management application.

## Prerequisites
- Node.js

## Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the application:
   ```bash
   npm run dev
   ```

3. Open your browser to the local server URL provided in the terminal.

## Features
- Sales and Invoicing
- Service Desk & Repairs Management
- Inventory Tracking
- RMA & Returns
- Expense Management
- Analytics & Reporting
